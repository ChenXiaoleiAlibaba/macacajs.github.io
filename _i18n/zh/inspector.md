## 安装

```shell
$ npm i app-inspector -g
```

## 用法

```shell
# 直接 -u + 设备的 udid 即可
$ app-inspector -u xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

## 帮助

```shell
$ app-inspector -h
```
