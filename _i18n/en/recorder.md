## Installation

```bash
$ npm i uirecorder -g
```

## Usage

```bash
# start macaca server
$ macaca server --port 4444 --verbose
# start recorder
$ uirecorder start --mobile
# macaca run
$ macaca run -p 4444 -d test.spec.js --verbose
```

inspired by [uirecorder](https://github.com/alibaba/uirecorder)
