# Recorder

## Installation

```shell
$ npm i uirecorder -g
```

## Usage

```shell
# start macaca server
$ macaca server --port 4444 --verbose
# start recorder
$ uirecorder start --mobile
# macaca run
$ macaca run -p 4444 -d test.spec.js --verbose
```
