上手指南

运行官方示例

将官方示例(mobile-app-sample-nodejs)克隆到本地，更多的示例请访问macaca-sample。

$ git clone https://github.com/macaca-sample/mobile-app-sample-nodejs.git --depth=1
$ cd mobile-app-sample-nodejs
$ npm i
# 更多运行方式见Makefile
$ macaca run --verbose

运行命令行工具

$ macaca run

更多的命令行用法请见命令行文档.
