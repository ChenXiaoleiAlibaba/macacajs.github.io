# Macaca docs site

## Running locally

Start a dev server at `localhost:4000`:

### install jekyll

```shell
$ sudo gem install jekyll
```

### install bundler

```shell
$ sudo gem install bundler
```

### bundle install

```shell
$ bundle install
```

###  start server

```shell
$ jekyll server
```

then you can open to see this website: [http://localhost:4000/](http://localhost:4000/)

## Build Site

```shell
$ jekyll build
```
